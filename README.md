# Hoop-club-website
A full-stack community platform for basketball enthusiasts to connect, share posts, and participate in events.(wip)

# Tech Stack

# Frontend
- React
- Vite
- Tailwind CSS
# Backend
- Node.js
- Express

# Database
- PostgreSQL (deployed through Neon)

# ORM
- Prisma

# Features

- User authentication (planned)
- Create and view posts
- Comment system
- Event management
- Responsive UI
